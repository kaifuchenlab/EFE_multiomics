# EFE_LV_multiomics_paper
Single cell multiomics data analysis for the EFE project

This repository contains the code and supplemental materials for the EFE multiomics project.

Supplemental Materials
File	Description
EFE_vs_CTRL_props.txt	Summary statistics comparing cell proportions between EFE (Children) and CTRL (Children) groups
EFE_vs_Fetal_props.txt	Summary statistics comparing cell proportions between EFE (Children) and CTRL (Fetal) groups
EFE_vs_Young_props.txt	Summary statistics comparing cell proportions between EFE (Children) and CTRL (Young) groups
4_Reviewer4_Q6_FigureE	Boxplot comparing cell proportions in EFE to age-matched CTRL
5_ReviewMaterials.xls	Detailed gene lists including: (1) MERFISH gene probes, (2) EC marker genes for Figure G, (3) fibrosis genes for Figure G
6_Methods_Online_030426.docx	Comprehensive Materials and Methods section for the study

Scripts
Script	Description
1_Sample_Integrations_030126.R	R code for integrating single nucleus multiomics data with public datasets
1_SingleCell_Analysis_030126.R	R code for single cell analysis
1_CellChat_GO_030126.R	R code for cell-cell communication analysis and Gene Ontology (GO) analysis
1_MERFISH_analysis_030126.ipynb	Python notebook for spatial transcriptomics data analysis, including cell type annotation and neighborhood enrichment
Data Overview
This repository supports the multiomics analysis of Endocardial Fibroelastosis (EFE), including:
•	Single nucleus multiomics integration
•	Cell type proportion comparisons across developmental stages
•	Spatial transcriptomics analysis (MERFISH)
•	Cell-cell communication networks
•	Gene ontology enrichment analysis
